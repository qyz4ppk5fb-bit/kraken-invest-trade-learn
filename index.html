Step-by-step install

  1. Download the ZIP using the button above.
  2. Unzip the folder (Windows: right-click → Extract All; macOS: double-click).
  3. Open the electron/dist/ folder and run the installer (.exe) or run the executable directly.
  4. Start the Flask server (if using the web version) by running ./run.sh or python app.py.

Troubleshooting

If Windows blocks the installer, right-click → Open and accept "More info" → Run anyway (this is standard for unsigned apps).
